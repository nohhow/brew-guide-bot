					- 강수량 : 비가 많이 오면 연수가 된다. 그리고 물이 더러워진다. 그러면 염소를 더 많이 투입하게 된다. 연수에 염소까지 많이 들어가기 때문에 물의 영향이 커질 수 있는 시기라는 점을 꼭 감안하자.

						- 온도에서 영향을 주는건 CO2 다. 대기 중에 있는 CO2 가 온도가 낮아지면 용해됨. 탄산 경도가 상승하게 된다면? 과하게 상승하면 "톤"이 어둡게 된다. 그렇다면 산미를 더 끄집어 내는 추출을 해야함.

# 브루잉 레시피 설계

가장 중요한 기준 : Tasting (맛)
	- 얼라인먼트(alignment) 기준 정렬을 맞춰서 칼리브레이션 할 수 있어야함.
	- 실제 측정 수치와 팀원들 또는 손님과의 맛 기준을 맞춰야함. 농도 1.2 면 다른 사람들은 이렇게 느끼는 구나. 이런걸 알아야함.
- 농도 
	- 쓴맛 (3) - 쓴맛을 사람들은 가장 많이 느끼게 됨 그래서 3이야. 가장 적게 뽑아내야함. 꼭 담아야하나? 밸런스를 위해서 담아야함. 신맛을 눌러주는 역할, 긍정적인 bitterness. 커피는 당연히 쓰다. 다 식고 나면 쓴거야. 근데 인지부조화를 이용해서 즐길 수 있게 설계하여 즐기는 거지. 바디감과 여운
	- 신맛 (2) - 매력 요소를 심어주는 느낌으로 뽑아내야함. 
	- 단맛 (1) - 가장 비중있게 뽑아내야함.
	- 쓴맛, 신맛, 단맛이 1:1:1 로 추출되었다고 가정했을 때, 농도가 높아지면?? 당연히 쓴맛을 많이 느끼니까 쓰게느껴진다. 농도를 조절하는 기준은 여기서 힌트를 얻으면 된다.
- 강도
	- 농도는 강도와 비례관계
	- 섬세한 커피를 만들고 싶으면 농도를 높이면 안됨.
- 수율
	- 한 컵 안에 얼마나 밀도있게 채울것이냐?
	- 커핑은 굉장히 동등하게 비율에 맞게 채워지잖아? 이게 판단 기준으로 잡기 좋고
	- 추출은 내가 제어하는 것에 따라서 다른 비율로 채워넣을 수 있다. 
	- 산미 : 풀, 향 : 꽃, 쓴맛 : 나무
		- 정원을 가꾼다고 생각해봐라. 
	- 수율이 높을 수록 여운이 길 확률이 높다.
		- 18%(낮은편) ~ 22%(높은편)
		- 코스피어는 20% ~ 20.5% 를 타겟함.
			- 발효빈은 수율을 좀 줄이는편
			- 워시드는 좀 더 높이는편
			- 22퍼 위로 올리려면 트리콜레이트나 넥스트브루어를 사용하여 논바이패스 추출하면됨.

- 농도와 수율
	- 농도가 1.2 보다 높으려면 수율은 낮아지는게 좋음. 둘다 높으면 짭짤하게 느껴짐
	- 1.2 보다 낮추려면 수율을 높여주는게 좋음. 연하니까 빼곡하게 채우자.


발효빈 특성
	- 향이 강하고, 쓴맛도 좀 높을 가능성 있음
		- 쓴맛을 억제하기 위해 콜드룸 발효를 하기도함. 이 때 부정적인 것들이 많이 사라짐
	- 드라잉 기간이 길다? 
		- 자연적 발효 기간을 가짐. 수분에 의한 변성이 길다고 생각하면 됨. 드라잉 기간이 길면 더 많이 녹아진다고 생각하면 될듯.
		- 즉, 빠르게 말렸다면 추출시 농도감 있게 가져가는게 좋음.

# 원재료의 특성

1. 산지별 특성 및 품종
	- 대만은 고도가 높은편이라서 거의 중남미 산지의 콩처럼 다뤄도됨
	- 아프리카 콩은 클로깅(병목)이 많이 일어남. 밀도는 높은데 경도가 낮아서 미분이 많이 생기는 편이다.
	- 중남미는 습도가 높은지역 콩일 수록 성분 추출이 잘 안되는 편. (온두라스, 니카라과?)
	- 밀도와 경도를 신경쓴다면
		- 그라인더 RPM을 변경할 수 있다면 미분 줄이기가 가능할것 
	- 다크로스팅과 라이트 로스팅
		- 라이트로스팅 원두 : 물빠짐이 안 돼서 최근에 빠르게 추출하는 도구들을 찾는다.
		- 다크로스팅 원두 : 라이트로스팅과 반대로 느리게 느리게 추출
		- 로스팅이 빠르면 ? : 다공질화가 되어 쉽게 스며들고 쉽게 빠져나감
			- 이는 즉 추출할 때 빠르다. 추출 타임이 빠를수밖에없음.
			- 그래서 이 특성에 맞게 물온도를 조절해주면 좋음. 느리게 추출되는 녀석은 물온도 낮춰주자.

2. 물에 따라 추출경향이 다르다.
	- 스케일이 생겼다 : 탄산경도와 미네랄/마그네슘/칼슘 의 결합체다. 
		- 시고 자극적인 느낌으로 추출될 수 있음.
		- 끓인물을 또 끓여서 쓰지말자!

# 블루밍(뜸, Blooming)의 이해 및 활용

준비단계이기도 하고, 1차추출이기도 하다.

시간 : 세포벽으로 도달하는 시간
물양 : 세포벽까지 필요한 양

불리면 불릴 수록 안 빠져나와도 되는 녀석들도 빠져나올 수 있다.
- 물양이 부족할 때 : 과소추출의 경향. 응집된 신맛이나 가벼운 쓴맛 도드라짐
- 물양이 많을 때 : 과다추출의 경향, 물맛이 강해지며 쓴맛의 강도가 높고, 떫어짐
- 시간이 부족할 때(저분자 성분 위주) : 향미의 강도가 부족하며, 신맛이나 쓴맛이 도드라짐
- 시간이 과도할 때(고분자 성분 위주) : 향미의 강도가 강하며, 쓴맛이 강도가 높고, 어두움

블루밍 가이드
	- 라이트 로스트 : 30~40초, 물양 2.5~4배수
	- 미디엄 로스트 : 30~35초, 물양 2~3배수
	- 다크 로스트 : 30~40초, 물양 1~2배수

## 📖 정리 노트

### 1. 원재료의 이해

#### 1-1. 산지별 특성

**대만**
- 고도가 높은 편 → 중남미 산지 콩처럼 다뤄도 무방

**아프리카**
- 클로깅(병목) 현상 빈번
- 밀도는 높지만 경도가 낮음 → 미분 생성이 많음
- 대응: 그라인더 RPM 조절 가능하다면 미분 감소 시도

**중남미**
- 습도가 높은 지역 원두일수록 성분 추출이 잘 안 되는 경향
  - 예: 온두라스, 니카라과

#### 1-2. 로스팅 레벨에 따른 특성

**라이트 로스팅**
- 물 빠짐이 안 됨 → 빠르게 추출하는 도구 선호
- 최근 트렌드: 빠른 추출 도구 활용

**다크 로스팅**
- 라이트와 반대: 느리게, 느리게 추출
- 다공질화로 물이 쉽게 스며들고 빠져나감

**로스팅 속도**
- 빠른 로스팅 → 다공질화 → 추출 타임 빨라짐
- 느리게 진행된 로스팅 → 추출 타임 느려짐
	- 대응 : 느리게 추출되는 원두는 물 온도 낮춤

#### 1-3. 발효빈 (Fermented Bean) 특성

**기본 특징**
- 향이 강함
- 쓴맛 높을 가능성
- 콜드룸 발효 시: 쓴맛 & 부정적 요소 억제

**드라잉 기간에 따른 전략**
- 드라잉 길면: 자연적 발효 기간 보유 → 수분 변성 오래 → 더 많이 녹아남
- 빠르게 말린 원두: 추출 시 농도감 있게 가져가는 것이 유리

**발효빈 최적 물 조건**
- 적정 ppm: 40ppm (연수 계열)
- 삼다수 (탄산경도 20~30ppm): 발효빈의 과한 특성 억제

---

### 2. 물의 특성과 계절별 변수

#### 2-1. 계절별 물의 변화

**봄 → 여름 (강수량 영향)**
- 비 많이 오면: 연수화 + 물 오염 → 염소 투입량 증가
- 연수 + 과다 염소 = 물의 영향 극대화
- 가장 예민한 시기

**가을 → 겨울 (온도 영향)**
- 습도 변화 → 원두 상태 변화
- 온도가 빈 온도에 영향 (여름: 셀러 보관 필요)
- 단순히 차가운 원두: 향미 발현 어려움
  - 얼어있는 원두: 곱게 갈거나 수온 높여서 대응 가능
- **CO2 용해도 변화**
  - 온도 ↓ → 대기 중 CO2 용해 ↑ → 탄산경도 ↑ → 톤 어두워짐
  - 대응: 산미를 더 끌어내는 추출 필요

#### 2-2. 적정 TDS & 탄산경도

| 원두 타입 | 적정 TDS(ppm) | 탄산경도 | 특징 |
|---------|-------------|---------|------|
| 일반 원두 | 70~90 | 30~35 | 표준 추출 |
| 발효빈 | 40 | - | 연수 계열 (50ppm 이하) |
| 삼다수 사용 | - | 20~30 | 발효빈의 과한 부분 억제 |

#### 2-3. 물 관련 주의사항

**스케일 생성 = 탄산경도 + 미네랄/마그네슘/칼슘 결합체**
- 시고 자극적인 느낌으로 추출될 수 있음
- **주의**: 끓인 물을 재가열하지 말 것

### 3. 코스피어(Cospir) 브랜드 철학

#### 3-1. 핵심 가치 (3대 요소)

1. **캐릭터 발현** - 원두 고유의 노트가 잘 나타나는가
2. **단맛** - 가장 비중 있게 추출
3. **클린컵** - 화질의 선명도처럼 깨끗한 표현

#### 3-2. 클린컵의 진정한 의미

- 아무 맛도 안 나는 것 ≠ 클린컵
- 캐릭터가 잘 나오면서 깨끗한 것 = 진정한 클린컵
- 하이엔드 커피: 클린컵이 점수 차별화의 핵심 포인트

#### 3-3. 필수 운영 원칙

- **기준 레시피 반드시 보유**
- 얼라인먼트(Alignment): 팀원/손님과 맛 기준 동기화
  - 예: "농도 1.2는 이런 느낌"을 공유

---

### 4. 브루잉 레시피 설계 원리

#### 4-1. 가장 중요한 기준: Tasting (맛)

**맛 성분별 추출 우선순위와 역할**

| 순위 | 맛 성분 | 추출 비중 | 역할 |
|-----|--------|---------|------|
| 1 | 단맛 | 가장 크게 | 기본 베이스 |
| 2 | 신맛 | 중간 | 매력 요소 심기 |
| 3 | 쓴맛 | 가장 작게 | 밸런스, 신맛 억제, 바디감·여운 |

**쓴맛의 긍정적 역할**
- 커피는 본질적으로 쓰다 (다 식으면 쓴맛 남음)
- 인지부조화를 이용해 즐길 수 있게 설계하는 것
- 밸런스를 위해 반드시 필요

**농도와 맛의 관계**
- 단맛:신맛:쓴맛 = 1:1:1 추출 시, 농도 ↑ → 쓴맛 강하게 인지 (역치)
- 섬세한 커피 원하면: 농도 낮춤
- 농도 ≈ 강도 (비례 관계)

#### 4-2. 수율(Extraction Yield) 설계

**수율의 의미**
- 한 컵에 얼마나 밀도 있게 채울 것인가?
- **정원 비유**: 산미(풀), 향(꽃), 쓴맛(나무) - 어떻게 정원을 가꿀 것인가?

**수율 범위 및 코스피어 타겟**

| 분류          | 수율 범위        | 특징                     |
| ----------- | ------------ | ---------------------- |
| 낮은 편        | 18%          | -                      |
| **코스피어 타겟** | **20~20.5%** | 기본 타겟                  |
| 코스피어 - 발효빈  | 20% 미만       | 수율 낮춤                  |
| 코스피어 - 워시드  | 20.5% 이상     | 수율 높임                  |
| 높은 편        | 22%          | 트리콜레이트/넥스트브루어 논바이패스 추출 |

**수율 특성**
- 수율 ↑ → 여운 길어질 확률 ↑

#### 4-3. 농도-수율 밸런싱 전략

| 농도 | 수율 조절 | 이유 |
|-----|---------|------|
| > 1.2 | 수율 ↓ | 둘 다 높으면 짭짤하게 느껴짐 |
| < 1.2 | 수율 ↑ | 연하니까 빼곡하게 채우자 |

### 5. 추출 도구(디바이스) 및 테크닉

#### 5-1. 드리퍼 선택 가이드

**드리퍼 형태**
- 포켓형태 / 콘형 / 플랫바텀
- 리브 많음 → 바이패스 많음
- 원두와 컨택타임 고려

**추출구 크기**
- 콘형: 추출구 좁은 게 좋음
- 플랫바텀: 홀 클수록 좋음

**추출 속도에 따른 특성**
- 빠른 추출: 직관적이고 섬세한 노트 강조 (초반)
- 느린 추출: 여운 길고 라운드한 커피

**드리퍼 재질**

| 재질           | 특성           | 열량 프로파일  | 활용                         |
| ------------ | ------------ | -------- | -------------------------- |
| 세라믹          | 열량 유지 가능     | 일직선 유지   | 92도 물 사용 시에도 커피베드 온도 일정 유지 |
| 세라믹 (차가운 상태) | 예열 안 함       | 점차 온도 상승 | 점차 온도 올리는 프로파일             |
| 메탈           | 빨리 가열, 빨리 식음 | 점차 온도 하강 | 점차 열량 줄이는 프로파일             |

#### 5-2. 추출 방식

**침출식**
- 저어주기 + 충분한 시간 반드시 필요
- 물을 붓는 대로 빠지지 않고 수위가 어느정도 유지된다 → 침출 비중 있는 것으로 볼 수 있음 (여과방식이라면 붓는 족족 바로 빠져야함)

**푸어링식**

| 드리퍼 형태 | 물줄기 | 여과 비율 | 바이패스 | 확산 효율 | 침출 비율 |
|----------|-------|---------|---------|----------|---------|
| 콘 | 가는 줄기 | ↑ | ↓ | ↓ | ↓ |
| 콘 | 굵은 줄기 | ↓ | ↑ | ↑ | ↑ |
| 플랫바텀 | 가는 줄기 | ↑ | ↓ | ↓ | ↓ |
| 플랫바텀 | 굵은 줄기 | ↓ | ↑ | ↑ | ↑ |

**푸어링 기본 원칙**
- 물길의 시작점과 끝점은 반드시 센터

**EK43 그라인더 특성**
- 의도적으로 부분적 미분 생성 (공식 엔지니어 오피셜)
- 미분 zone이 별도로 존재
- 미분 활용 → 클로깅 유도 → 바이패스 감소 + 침출 비중 증가

#### 5-3. 변수별 조절 가이드

**Ratio (원두:물 비율)**
- 하이엔드 커피일수록: 물 비율 ↑
- 마지노선: 원하는 농도
- 물 많이 부을수록: 스펙트럼 넓어짐
- **코스피어 스타일**
  - 어두운 커피: 물 ↓ (더 어둡게)
  - 밝은 커피: 물 ↑ (더 밝게)

**도징량**
- 도징량 많음 → 온도 유지 잘 됨 → 물 온도 낮춰야 함

**물 온도**
- 쓴맛, 단맛, 신맛의 비중이 1도에 따라 달라짐
- 세밀한 조절 필요

**나눠 붓는 횟수**
- 나눠 붓기 = 확산 효율 높이기
- 나눠 붓는 횟수 ↑ → 물 온도 ↓

**분쇄도**

| 목표          | 분쇄도 | 비고                   |
| ----------- | --- | -------------------- |
| 밝고 섬세한 커피   | 굵게  | -                    |
| 묵직하고 강렬한 커피 | 곱게  | 김승백 바리스타 선호 (단중밸 중시) |
| 품질 낮은 원두    | 굵게  | -                    |
| 품질 좋은 원두    | 곱게  | -                    |

*"더 굵게/더 곱게"는 클릭수 기준 1칸 차이 정도

**분쇄날 타입**
- 플랫버(Flat Burr): 섬세함, 클린컵
- 코니컬버(Conical Burr): 복합적

**그라인더 RPM**
- 낮음: 명확함
- 높음: 복합성

### 6. 블루밍(Blooming, 뜸) 가이드

#### 6-1. 블루밍의 의미

- 준비 단계이자 1차 추출
- **시간**: 세포벽으로 도달하는 시간
- **물양**: 세포벽까지 필요한 양

#### 6-2. 물양과 시간에 따른 영향

**물양**

| 상태 | 경향 | 특징 |
|-----|------|------|
| 부족 | 과소추출 | 응집된 신맛, 가벼운 쓴맛 도드라짐 |
| 과다 | 과다추출 | 물맛 강함, 쓴맛 강도 높음, 떫음 |

**시간**

| 상태 | 추출 성분 | 특징 |
|-----|---------|------|
| 부족 | 저분자 위주 | 향미 강도 부족, 신맛/쓴맛 도드라짐 |
| 과도 | 고분자 위주 | 향미 강도 강함, 쓴맛 강도 높음, 어두움 |

**주의**: 불리면 불릴수록 안 빠져나와도 되는 성분도 빠져나올 수 있음

#### 6-3. 로스팅 레벨별 블루밍 가이드

| 로스팅 레벨 | 시간 | 물양 (배수) |
|----------|------|-----------|
| 라이트 로스트 | 30~40초 | 2.5~4배 |
| 미디엄 로스트 | 30~35초 | 2~3배 |
| 다크 로스트 | 30~40초 | 1~2배 |

*불림시간 30초 아래로는 추출 테크닉이 너무 어려워짐

---

### 7. 커핑 & 평가 체크리스트

#### 7-1. 평가 기준

**캐릭터 (Character)**
- 원두의 고유 노트가 잘 나타나는가?
- 프래그런스(Fragrance) 강도 - 어떤 맛이 날지 유추 가능
- 웻 아로마(Wet Aroma) 체크
- *프래그런스와 웻 아로마가 가장 단서가 많음*

**농도 (Strength)**
- 직관적으로 파악 가능
- 농도가 빨리 생기는가?
- 기준 레시피 있다면 커핑보다 기준 레시피로 확인 권장

#### 7-2. 시간대별 평가 (10~50분)

| 시간 | 평가 포인트 |
|-----|-----------|
| 10분 | 초기 캐릭터 확인 |
| 20분 | 중반 밸런스 확인 |
| 30분 | 일반 원두 마지노선 |
| 40분 | 일반 원두는 버티기 힘든 구간 |
| 50분 | **하이엔드 기준**: 여전히 깨끗하고 긍정적인가? |

**평가 기준**
- 떨어진 온도에서도 긍정적 = 좋은 커피
- 시간에 따른 톤 변화 확인
- 한 가지 톤 쭉 유지 = 유니크한 특성

### 변수 통제 매트릭스 (Variable Control Matrix)

#### A. 원두 특성에 따른 기본 세팅

| 원두 특성 | 분쇄도 | 물온도 | 수율 타겟 | TDS(ppm) | 비고 |
|---------|-------|--------|---------|---------|------|
| 발효빈 | 기준 | 기준 | 20% 미만 | 40 | 연수 계열 물 사용 |
| 워시드 | 기준 | 기준 | 20.5% 이상 | 70~90 | 탄산경도 30~35 |
| 라이트 로스팅 | 굵게 | 높게 | 기준 | 70~90 | 빠른 추출 도구 |
| 다크 로스팅 | 곱게 | 낮게 | 기준 | 70~90 | 느린 추출 |
| 아프리카 | 기준~굵게 | 기준 | 기준 | 70~90 | 미분 주의, RPM 조절 |
| 중남미 (습한 지역) | 곱게 | 높게 | 기준 | 70~90 | 성분 추출 어려움 |
| 하이엔드 | 곱게 | 기준 | 20.5% | 70~90 | 물 비율 ↑, 클린컵 중시 |

#### B. 맛 문제 발생 시 조절 가이드

| 문제 증상 | 원인 추정 | 조절 방향 |
|---------|---------|---------|
| 쓴맛 강함 | 과다추출 / 농도 높음 | 분쇄도 굵게 / 물온도 ↓ / 물 비율 ↑ / 수율 ↓ |
| 신맛 강함 | 과소추출 / 탄산경도 높음 | 분쇄도 곱게 / 물온도 ↑ / 산미 끌어내는 추출 |
| 단맛 부족 | 추출 불균형 | 블루밍 시간/물양 조절 / 농도 재조정 |
| 짭짤함 | 농도 ↑ + 수율 ↑ | 수율 ↓ 또는 물 비율 ↑ |
| 연하고 밍밍함 | 농도 ↓ + 수율 ↓ | 수율 ↑ (빼곡하게 채우기) |
| 향미 강도 부족 | 블루밍 시간 부족 | 블루밍 시간 늘리기 (고분자 추출) |
| 과도한 쓴맛/어두움 | 블루밍 과다 | 블루밍 시간 줄이기 / 물양 줄이기 |
| 톤이 어두움 (겨울) | 탄산경도 ↑ (CO2 용해) | 산미 강조 추출 / 물온도 조절 |
| 시고 자극적 (스케일) | 탄산경도 + 미네랄 결합 | 물 교체 / 끓인 물 재사용 금지 |

#### C. 농도별 추출 전략

| 목표 농도 | 수율 전략 | 맛 밸런스 전략 | 어울리는 원두 |
|---------|---------|-------------|------------|
| > 1.2 | 낮춤 | 쓴맛 주의, 단맛 강조 | 묵직한 커피 선호 시 |
| 1.2 | 기준 | 밸런스 중시 | 일반적 기준 |
| < 1.2 | 높임 | 빼곡하게 채우기, 여운 강화 | 섬세한 커피, 하이엔드 |

#### D. 계절별 체크리스트

| 계절 | 주요 변수 | 체크 포인트 | 대응 방법 |
|-----|---------|-----------|---------|
| 봄→여름 | 강수량 | 연수화, 염소 증가 | 물 TDS 측정, 필터 교체 주기 단축 |
| 여름 | 원두 온도 | 셀러 보관 여부 | 단순 차가움 주의, 예열 고려 |
| 가을→겨울 | 온도, 습도 | CO2 용해, 탄산경도 ↑ | 산미 강조 추출, 원두 보관 온도 관리 |
| 겨울 | 원두 상태 | 얼어있거나 차가움 | 분쇄도 곱게 / 수온 ↑ / 향미 발현 주의 |

#### E. 드리퍼별 추출 세팅

| 드리퍼 | 물줄기 | 추출 속도 | 어울리는 커피 | 주의사항 |
|-------|-------|---------|------------|---------|
| 콘형 (좁은 추출구) | 가는 줄기 | 느림 | 여운 길고 라운드한 커피 | 바이패스 적음 |
| 콘형 (좁은 추출구) | 굵은 줄기 | 중간 | 밸런스형 | 침출 비율 ↑ |
| 플랫바텀 (큰 홀) | 가는 줄기 | 중간 | 섬세하고 깔끔한 커피 | 확산 효율 낮음 |
| 플랫바텀 (큰 홀) | 굵은 줄기 | 빠름 | 직관적이고 밝은 커피 | 바이패스 많음 |

#### F. 코스피어 스타일 가이드

| 추구하는 맛 | 설정 |
|----------|------|
| **농도** | 1.2 기준 (섬세함 추구 시 낮춤) |
| **수율** | 20~20.5% (발효빈은 낮춤, 워시드는 높임) |
| **맛 비율** | 단맛(1) > 신맛(2) > 쓴맛(3) |
| **핵심 가치** | 캐릭터 발현 + 단맛 + 클린컵 |
| **Ratio** | 밝은 커피: 물 ↑ / 어두운 커피: 물 ↓ |
